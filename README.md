# Exchange-App
MVP
开发背景：
1.随着社会不断进步，人民生活水平不断提高，生活物品日益丰富，人们逐渐累积了大量冗余物品。与此同时，又对不拥有的物品存在着使用需求，导致不断购买和累积，造成了物品使用的极大浪费，也使得生活成本不断提高。此时，人与人之间的冗余物品相互交换利用，不仅充分利用了物品价值，有效节约社会资源，降低的成本还可以使人们拥有更多机会去尝试和使用现代社会中的琳琅满目的商品，在一定程度上提高了人们的生活水平。
2. 如今移动互联网已普及到日常生活，人与人间交流更加便捷、高效，使得更宽广范围内的物品交换行为易于实施。
3.当前已存在很多二手物品交易平台，但都普遍存在以下问题：首先， 物品繁杂，针对大众群体，很难根据特定人群对物品进行分类；其次，不同地域之间运输成本高，远途交换无法保证质量

需求分析：
按照功能模块划分：
0.欢迎界面
1.登陆和注册
2.首页
3.物品页
4.账户中心
5.朋友圈

详细功能模块规划现用今目标软件管理：https://web.jingoal.com/#/igoal/detail/3999905?active=all&refferkey=igoal-list-all&_k=07spil

应用开发：
分为客户端和服务器端，客户端采用Android语言开发，服务端使用SpringBoot框架开发，数据库管理使用JPA。
